# 🚀 DigiTools

## 📖 Description

DigiTools is a responsive React web application that showcases premium digital tools. Users can browse products, add them to a cart, and simulate a checkout experience with a clean, modern UI based on a Figma design.

---

## ⚙️ Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* React Toastify
* JSON

---

## ✨ Features

* Add to cart functionality
* Remove items from cart
* Cart item count in navbar
* Total price calculation
* Prevent duplicate product addition
* Toast notifications for actions
* Responsive design across devices
